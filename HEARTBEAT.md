# HEARTBEAT.md - Periodic Checklist

This file is read on heartbeat polls (~30 minutes). Use it as a living checklist of things to check, refresh, or think about.

Add items that matter. Keep it small — token budget matters.

Examples:
- Check email for urgent messages
- Scan calendar for upcoming events (next 24-48h)
- Review today's open threads
- Look for relationship check-ins that are overdue

Leave empty if nothing is urgent. You'll respond `HEARTBEAT_OK` and stay quiet.
