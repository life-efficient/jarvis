# TOOLS.md - Local Config & Secrets

Store environment-specific details here:
- API keys (if not using env vars)
- Device names (camera, microphone, SSH hosts)
- Phone numbers, email addresses
- TTS voice preferences
- Local paths and shortcuts

Example:

```
OPENAI_API_KEY = sk-...
PRIMARY_PHONE = +1234567890
TTS_VOICE = nova
CAMERA_KITCHEN = 192.168.1.100
SSH_WORK_SERVER = user@work.example.com
```
